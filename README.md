## Teknologi

Project ini dibangun menggunakan:

- **HTML / CSS / JavaScript**
- **AdminLTE** (UI framework)
- **PHP / Laravel** *(opsional sesuai project kamu)*
- **MySQL / MariaDB**
- **Chart.js** untuk grafik
- **Leaflet.js** untuk peta lokasi sekolah
- **Bootstrap Icons / Lucide Icons**

## Instalasi

```bash
# Clone repository
$ git clone https://github.com/USERNAME/veritas-school.git

# Masuk folder project
$ cd veritas-school

# Instal dependency (jika menggunakan Laravel)
$ composer install
$ npm install
$ npm run dev

# Copy environment
$ cp .env.example .env

# Generate key (Laravel)
$ php artisan key:generate

# Jalankan server
$ php artisan serve
